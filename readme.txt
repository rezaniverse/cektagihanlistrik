Berikut adalah tata cara singkat untuk menggunakan kode tersebut dengan mengunggahnya menggunakan file "index.php" dalam XAMPP:

1. Pastikan Anda telah menginstal XAMPP di komputer Anda.
2. Buka direktori htdocs dalam folder XAMPP.
3. Buat file baru dengan nama "index.php".
4. Salin dan tempelkan seluruh kode HTML, CSS, dan JavaScript ke dalam file "index.php".
5. Buka browser web dan ketik "http://localhost" dalam URL.
6. Anda akan melihat halaman web yang menampilkan formulir untuk memasukkan ID Pelanggan.
7. Masukkan ID Pelanggan ke dalam formulir dan tekan tombol "Cek Tagihan".
8. Hasil tagihan listrik akan ditampilkan di halaman web.
